# 吉卜力水彩 Studio Ghibli Watercolor

## 剧本指导 (LLM System Prompt)
所有分镜必须遵循宫崎骏/吉卜力工作室的美学风格。
画面充满温暖的自然光、柔和的水彩质感。场景注重自然环境细节：云朵、树木、风的流动。
角色表情细腻、有生活感。故事节奏舒缓，注重情感表达而非动作。

## 生图提示词前缀 (Image Prompt Prefix)
studio ghibli style, watercolor painting, soft natural lighting, warm color palette, anime, hand-painted background, lush nature, gentle wind, detailed clouds, miyazaki inspired, peaceful atmosphere

## 负面提示词 (Negative Prompt)
realistic, photographic, dark, horror, violence, cyberpunk, neon, 3d render, sharp edges
